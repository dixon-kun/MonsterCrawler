The ADP Small Business Services (SBS) team is looking for an API Architect to help define the Service Oriented Architecture of its next generation products. 
We are looking for an API Designer/Data Architect to work on cutting edge projects using the latest tech such as NodeJS. This person will be responsible for designing new APIs and working hands on with the lean/agile teams implementing them. These activities include, but are not limited to Domain Model creation and maintenance, API schema and development and specifications/documentation for complex ADP products across platforms. 
We strive for every interaction to be driven by our CORE values: Insightful Expertise, Integrity is Everything, Service Excellence, Inspiring Innovation, Each Person Counts, Results-Driven, &amp; Social Responsibility. 
RESPONSIBILITIES: 
&#183; Create and maintain Domain Models as part of the SBS Architecture/Development team 
&#183; Work as part of fast paced innovation/development teams to design new APIs and evolve them during POCs and implementation 
&#183; Evangelize, talk, train and generally show love for APIs. 
&#183; Maintain API registry for legacy/new APIs across the product set and provide input into API planning/strategy/roadmap 
&#183; Design and implement our Microservice CI/CD pipeline, test strategy, infrastructure architecture, monitoring, alerting, tracing, etc. 
&#183; Performance tuning, from testing and monitoring to caching strategies 
&#183; Define security models and managing Pen Testing schedules and prioritizing findings 
&#183; Jump into code as needed to ensure important team commitments are met. 
Candidate Profile: 
 &#183; Proficiency translating complex solutions into UML diagrams into Enterprise Architect or whatever tool you love 
&#183; Experience architecting the API ecosystem: infrastructure, networking, security, Microservice management, deployment, service discovery, as well as tracing, debugging, monitoring, alerting, etc. 
&#183; Experience designing and developing highly available, performant, cacheable APIs 
&#183; Extensive experience with at least one modern programming language, such as NodeJS (preferred), C#, Java, Go, Python. 
&#183; Working knowledge of tools such as Jenkins and Docker 
&#183; Passionate advocate for TDD and best software development practices 
&#183; Strong problem solving, analytical skills and interpersonal skills. 
&#183; Strong project and time management skills including multi-tasking in fast-paced development environments. 
&#183; Must have an expensive understanding and proven working record of HTTP and REST.

&#183; Must be comfortable and enjoy documenting their APIs and discussing and educating others (product, developers etc) 
&#183; Must be able to lead discussions between domain experts to extract the necessary information to develop an accurate model.

&#183; Four-year college degree in Computer Science or Engineering or related discipline preferred. Equivalent experience accepted.
