This is a position within the Wunderman Product Development team. This individual works in a Support/development role as part of a development team designing/supporting software products and services that will operate as part of the Consumer Engagement Platform (CEP), which supports clients’ digital marketing activities. Software solutions include creating and supporting web services transactional processes, integrating multiple 3 rd party digital marketing platforms and tools, supporting large data movement processes, matching processes and data delivery.


  • Supports existing applications with troubleshooting and/or enhancements
  • Support automated release management
  • Diagnoses and corrects system problems
  • Assist in QA of development processes for web services / Java development
  • Creates documentation in accordance with established processes and policies

  • Bachelor’s Degree in Computer Science, Management Information Systems, Mathematics, Engineering, or equivalent
  • Two years relevant job experience
  • Interest in learning the concepts and practices, acquiring the skills, and developing the expertise needed to succeed in this position
  • Experience with JSON and XML is required
  • Must possess good interpersonal skills
  • Must possess good communication skills in both verbal and written English
  • Must have experience in core competencies, Java and Database skills
  • Must have experience with SQL statements and with stored procedures and triggers
  • Experience in Java Application servers and JEE development is required
  • Experience in REST and SOAP web services is a plus
  • Experience with big data is a plus
  • Experience with web services, glassfish application server, Jenkins, Maven, SVN and/or other Apache open source projects is a plus
