Data Scientist, Mid

Key Role:
Apply programming expertise to part of an Air Force client’s data science team. Maintain responsibility for developing a data science capability that supports US Air Force intelligence, Surveillance, and Reconnaissance platforms. Analyze large data sets and create tailored, repeatable, and scalable approaches for data science processes that range from data preparation to follow-up analysis. Leverage database architecture, middleware, and fundamental UI design and research and nominate open information sources for data mining to support ongoing analytical efforts. Conduct systems programming and support activities, including writing and revising segments of language code or processing while documenting developed processes and assisting with the development of training materials designed to teach military intelligence analysts the programming and mathematical facets of data science for developed processes. This position is located in Offutt Air Force Base, NE.

Basic Qualifications:
-Experience in working with multiple programming languages, including Python, PHP, Java, or C#
-Ability to effectively analyze and synthesize large amounts of complex data into insights
-Ability to work both as part of a team and independently
-Ability to identify and solve problems
-TS/SCI clearance with a polygraph
-BA or BS degree in CS, IT, Engineering, Information Systems, Mathematics,  or Data Science

Additional Qualifications:
-Experience with natural language processing (NLP) and machine learning (ML) tools and code libraries
-Experience with Agile development practices, including JIRA
-Experience with using Git version control
-Experience in working with geospatial data, incuding Comma Separated Value (csv) files, Keyhole Markup Language (kml), and Esri shapefiles
-Experience with the intelligence community (IC) or a foreign language
-Experience with gaming or advanced visualization
-Ability to implement projects using Cloud architecture, including AWS and OpenShift
-MA or MS degree in CS, Data Science, Software Engineering, Systems Engineering, Predictive Analytics, Data Mining, Statistics, or related field

Clearance:
Applicants selected will be subject to a security investigation and may need to meet eligibility requirements for access to classified information; TS/SCI clearance with polygraph is required.
