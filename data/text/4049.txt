          Tasks may include
           Establish and implement end to end proof of concepts and analytic applications
           Create innovative methodologies for extracting value from data

           Work with open source tools in the Hadoop ecosystem
           Program and analyse data using technologies and languages like Spark, Hive, Python, Scala, R, Scala and&#160; Java

           Strong programming and analytic skills
           Experience handling and analyzing time series data

             Experience with Hadoop ecosystem tools such as Hive and Spark etc.
             Experience with languages and technologies such as Spark, Hive, Python, R and&#160; Java
             Experience with scalable noSQL databases, time series databases and search engines (like MongoDB, OpenTSDB, Elasticsearch, etc.)
             Experienced user of Unix/Linux and the corresponding standard command line tools
             Knowledge and experience of microservice architecture style
             Experience developing and consuming RESTful API services
             Experience using code versioning systems (like Git)
             Experience in building and leading high-performing teams
