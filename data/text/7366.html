Database Engineer

Key Role:

Provide technical expertise for database design, development, implementation, information storage and retrieval, data flow, and analysis. Develop relational or NoSQL databases, use and develop Extract, Transform, and Load (ETL) tools, and project long-range requirements for database administration and design. Maintain responsibility for developing a database structure that fits into the overall architecture of the analysis and make trades among data volumes, storage, and security. Work closely with system owners to comprehend the schema and format of authoritative data sources and data scientists to comprehend the schema of database for analysis. Assist with recommendations for and the analysis and evaluation of systems improvements, optimization, and development or maintenance efforts. Help with feature engineering for data analysis by helping create ad hoc queries, scripts, and macros, update existing queries, create new ones to manipulate data into a master file, and build complex datasets using queries, tables, open database connectivity, and database storage and retrieval.

Basic Qualifications:

-2+ years of experience as a database engineer

-2+ years of experience with database design, development, implementation, information storage, and retrieval

-Experience with SQL at an expert level and creating ad hoc queries, scripts, and macros

-Experience with supporting analytical usage of databases

-Experience with database administration or engineering with Postgres, Oracle, MySQL, or SQL Server

-Secret clearance

-BA or BS degree

Additional Qualifications:

-Experience with ETL and creating data warehouse tables

-Experience with one or more NoSQL databases, including HBase

-Experience with the design, development, tuning, and creation of database environments

-BA or BS degree in a technical field preferred

Clearance:

Applicants selected will be subject to a security investigation and may need to meet eligibility requirements for access to classified information; Secret clearance is required.
