Senior Interactive Developer will be responsible for creating graphically

intensive user interfaces while using out of the box thinking to create new

techniques to visualize data. Using bleeding edge technology, the ideal

implementation of new applications that turn massive real-time datasets from

Create highly responsive front end pages

Visualize Data

Work with business users to gather functional requirements.

Work with client architect/senior developers to do high

level/low level design/architecture.

Coding and implementation using best practices to produce

high performing/scalable/secured modules.

Unit Testing/Integration Testing.

Code Configuration and Release Management.

Production Support and Bug Fixes.

REQUIREMENTS:

6 years of professional software development experience

Expert level of experience with JavaScript and

frameworks such as React.JS,

Bootstrap.JS, Node JS,

Experience on code bases greater than 2 million lines of code

Understanding of Functional Programming Concepts

Ability to visualize data / data science experience

Knowledge and experience with Adobe Illustrator

Relational SQL experience with large data sets, views and

stored procedures

Document Database Experience using Map Reduce

Data modeling and data warehouse experience

Ability to start green field projects and propose designs

Experience architecting high performance multi-tier systems

Experience writing code that deals with extremely large sets

of data

HTML5, CSS, LESS

C#, any scripting language

Queuing technologies: Rabbit, IBM MQ, ZeroMQ, MSMS MQ, Signal
