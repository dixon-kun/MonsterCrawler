GENERAL DESCRIPTIONEnsure that web-based software, automated reports, and supporting databases are implemented and enhanced as required.

SPECIFIC RESPONSIBILITIES:

1.Analyze systems to define testing approach to be used while considering all application dependencies.

Environments: C#/.NET Framework, MS SQL Server, TFS

Technologies: T-SQL, ADO.NET, Entity Framework, WCF, Web API, ASP.NET MVC, ASP.NET Web Forms, JavaScript

2.Implement and maintain unit test suites for existing codebases.

Environments: C#/.NET Framework, MS SQL Server, TFS

Technologies: NUnit/XUnit, NSubstitute/moq, T-SQL, ASP.NET MVC, ASP.NET Web Forms, JavaScript

3.Automate functional testing for existing applications.

Environment: Selenium

Technology: JavaScript, C#

4.Create Test Cases from User Stories and ensure new functionality meets Acceptance Criteria.

Environments: Custom Client-facing Research Applications, TFS

CURRENT & UPCOMING PROJECTS:

CBRE client-facing research application development projects are increasingly executed using an Agile approach. The development team currently supports, enhances, and implements new functionality for applications with stakeholders based in Boston, Dallas, Atlanta, and London.

Currently under development are a set of interactive, analytical widgets: a heat map widget, a motion-chart market comparison widget, and a complex market forecasting widget. Although embeddable in other applications and sites, these widgets will be spotlighted in a recently rebuilt client-facing portal.
