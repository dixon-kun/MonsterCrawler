Job Description

The Database Programmer is responsible for the coding/documenting of the Data Layer, Database Architecture, and EDI of the Test, Development, Staging and Production systems.


The Database Programmer’s Key Responsibilities include:

  • Be able to use the latest versions of: SQL Server Management Studio (SSMS) and Windows Server. Know SQL Server Integration Services (SSIS) and SQL Server Report Services (SSRS). Understand Master Data Management (MDM) and Master Data Services (MDS).
  • Create, maintain, and design company production data and DBMS based products. This includes SQL coding, planning and documenting the system.
  • Create, maintain, and design company production data and DBMS based products. This includes SQL coding, planning and documenting the system and its operation.
  • Create scripts and SSIS packages for data integration (SSIS), data maintenance or bug fixes.
  • Write SQL statements for report generation (SSRS).
  • Fundamental skills in data analysis and data modeling.
  • Creates the necessary technical and architectural documentation.
  • Assist in conducting analysis for the development of new systems and modifications of existing systems.
  • Ensure the integrity and security of the data.
  • Provide support to end users and customers with software-related issues or problems as necessary.
  • Ability to work in a team environment and/or mentor as needed.
  • Support the goals of the Information Systems Division and participate as a member of the Information Systems team.
  • Understand and operate the Opalis Integration Server (System Center - Orchestrator), understand the process of EDI communication via FTP and understand MDM with Profisee Maestro tools.
Qualifications

Required Qualifications:
Bachelor degree in Computer Science or a related discipline
Well-developed skills in a Database programming language with emphasis on MS SQL Server (T-SQL)
Minimum 1 year of experience as a programmer in a corporate environment.

Preferred Qualifications:
· Well-developed skills with SQL Server and/or MDM
· Solid and proven experience with complex queries
· Well-developed skills in a major operating system, Windows Server
· Well-Develop skills in a major programming language (VB, C#)
· Microsoft Certification
· MBA in a Data Science field
