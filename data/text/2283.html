Description

Do you want to be part of a team that is at the forefront of identifying potential vehicle safety issues though data and analytics? This is a great opportunity to apply your technical and business skills in a way that will benefitour customers.


In this role you will assist data scientists with tool and platform issues and directly interface with IT.

  • Work with IT and data scientists to ensure sandbox for text analytics is meeting our data scientist’s needs.
  • Ensure all Vehicle Safety environments are meeting our data quality metrics

·Specific Tool Support – Ensure that each data scientist tool set is operating efficiently.Would be responsible for developing a standard pattern for loading software including documentation on how everything gets set up such as connecting to servers, etc.


·Become tool SME for text classification and search tool and all associated processes to get data available for statistical analysis

  • Develop new ways to load, clean and standardize data

Behaviors/Other Skills

  • Flexible, can-do attitude, communication skills, business acumen, team player

Qualifications

·Bachelor’s degree in Software Engineering, Computer Science, Information Technology or other technical degree.


·Two or more years of work experience with software packages, open source software, databases and analytic tools, including Knowledge of data structures, data base design, data warehouses, business intelligence/analytic tools, ETL tools, and data integration methods.


·Experience dealing with unstructured data


· Experience with R or Python


·Strong organizational skills


·Strong oral and written communication skills


·High level of analytical ability and attention to details


·High level of personal integrity


·Ability and willingness to work in team-focused work environment
