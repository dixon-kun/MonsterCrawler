Position Description
In this role you will:

  • Build advanced machine learning models and improve the accuracy of those models
  • Crunch terabytes of data and build models from large dataset
  • Deploy the models in production where it will serve customers in real time
  • Work alongside senior machine learning researchers and define the right solution to implement
  • Put on the hacker's hat quickly convert an idea to a demonstration
  • Present your work to the internal and external community

Minimum Qualifications

  • PhD or masters in computer science with 2+ years of experience
  • Highly proficient in Python or Java
  • Experience in using libraries like scikit learn, tensorflow, nltk
  • Thorough understanding of the internals of the state of the art Machine Learning techniques
  • Familiarity with SQL like language and ability to crunch data
  • A good portfolio of Machine Learning projects
  • A github account

Additional Preferred Qualifications

  • Internship or industry experience with Machine Learning projects
  • Experience in handling large dataset
  • Familiarity with Deep Learning
