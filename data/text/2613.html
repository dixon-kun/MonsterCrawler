Qualifications
CACI is seeking a talented and self-motivated Software Developer to join an exciting, complex, and mission-focused team of professionals based in the Northern Virginia region. This position requires a current TS/SCI FSP .
Duties and Responsibilities:
Team members work in a multitasking, quick-paced, dynamic, process-improvement environment that requires experience with the principles of biometric technology, large-scale database and application development, large-scale file manipulation, data modeling, data mapping, data testing, data quality, and documentation preparation. The division directly supports data scientists and analysts by designing and engineering systems to meet exponential increases in digital data. Emphasis will be on both Java and C++ development tasks relating to image and video processing and large-scale data processing workflows.
Required Qualifications:
Typically requires bachelor’s degree or equivalent, and 16 to 18 years of related experience.
• On-the-job experience with Java EE development
• On-the-job experience with C++ development
• On-the-job experience implementing Java Native Interface (JNI) for C++ integration into Java
• On-the-job experience with the Linux operating system as necessary to perform server-side development tasks (i.e. build, configure, shell scripting) and also troubleshoot memory management and performance problems within C++
• On-the-job development experience with software development infrastructure tools for code management such as Eclipse, Ant, or Git/Subversion.
• Candidate can have experience in one of the applications or another similar application to meet the requirement.
• On-the-job experience with relational or NOSQL databases including one of Oracle, Postgres, Apache Solr, HBase, or Amazon Web Services offerings
Desired Qualifications:
• On-the-job experience with biometric identity data services and systems
• On-the-job experience developing or integrating with Software Developer Kits for biometrics
• On-the-job experience building software systems designed for scalability using Amazon Web Services
• On-the-job experience with Apache ActiveMQ or Camel
Company Description:
CACI employs a diverse range of talent to create an environment that fuels innovation and fosters continuous improvement and success. Join CACI, where you will have the opportunity to make an immediate impact by providing information solutions and services in support of national security missions and government transformation for Intelligence, Defense, and Federal Civilian clients. A member of the Fortune 1000 Largest Companies and the Russell 2000 Index, CACI provides dynamic careers for approximately 15,000 employees working in over 120 offices worldwide.
CACI is an Equal Opportunity Employer - Females/Minorities/Protected Veterans/Individuals with Disabilities.
