Description
This position is primarily responsible for running data science activities by using parallelized numerical/statistical computation using Hive, Spark, MapReduce, and/or other comparable big data technologies within the Hadoop framework to advance the company’s analytical projects. Preferred experience with predictive analytics using machine learning and presenting results with data visualization techniques to tell the story within the data.

Accountabilities:
- Analytics:
- Applies numerical/statistical techniques to gather and analyze information skillfully.
- Develops standardized, interpretable solutions and improvises when necessary.
- Curates visualizations of analytical results for communicating with wide range of audiences.
- Code Development:
- Develops and tests record linkage and feature selection processes over several joined data sets of more than 100 million unique records and 1500 unique columns.
- Implement fuzzy string matching and predictive modeling in Hadoop.
- Automates complex processes and deploys solutions using a version control system.
- Required knowledge of either scala, Java, or python with preferred knowledge of MS-SQL and R.
- Processing Optimization:
- Spearheads development of big data analytical programming within Hadoop cluster.
- Optimize cluster’s management of processing tasks to push efficiency of analytical and processing activities.
- Maintains organization of large data sets using Hadoop technologies such as Spark, Hive, SparkSQL, MapReduce, etc.
- Knowledge Transfer:
- Documents and presents findings to internal stakeholders.
- Interacts with other internal groups within Datamyx, especially data science, data engineering, and application development.
- Shares knowledge with other members of these teams and takes lead on Hadoop-specific projects.
