Software Engineer in Test

DataXu’s vision is to make marketing better using data science. The world’s top brands and agencies partner with DataXu to better understand and engage their customers across all available media formats, devices, channels, and buying modes.  Our advertising solution, ranked #1 in the industry by Forrester Research in 2015, provides marketers with unparalleled Media Activation, Marketing Analytics and Data Management capabilities.

We are seeking  software engineers with with strong quality testing skills to to help build the next generation or our business, Programmatic TV. Data Xu seeks a highly motivated test engineer to apply their problems solving skills in Data Xu' s automated media decision platform for advertisers utilizing Hadoop with Map Reduce. If you have a passion for working with large data (100s of Terabytes), this is the job for you!!!

On the Job

  • Design and implement automated tests for a highly distributed data analytics system and front-end web user interface.
  • Be an advocate for best practices in quality engineering
  • Develop reusable tools and libraries for functional and performance testing
  • Analyze User Stories and create testing strategies to optimize manual, tool-assisted, and fully automated tests.
  • Integrate tests with continuous integration build and deployment tools.
  • Work closely in engineering Scrum and Kanban teams to test new functionality, review unit test development and ensure testable interfaces are provided for new features and components
  • Collaborate with engineering and technical operation teams on all aspects of product validation
  • Evaluate and introduce new technologies, tools, and techniques to improve our software.

About You…

  • Strong programming skills with experience in object oriented development and x-unit test frameworks
  • Experience with one or more dynamic programming languages such as Ruby, Python, or JavaScript.
  • Strong experience with SQL (distributed database systems a plus)
  • Familiar with source control systems (git/GitHub, hg, svn), continuous integration tools (Jenkins, CruiseControl), and programming in a team environment
  • Proven experience writing API or web service tests
  • Experience with automated web UI testing using Selenium or similar tools highly desired
  • Highly proficient in Unix/Linux environments
  • Ability to work in a fast-paced startup environment with minimal supervision
  • Prior experience in Lean/Agile development environments preferred
  • Degree in Computer Science, Math, an Engineering discipline or equivalent relevant job experience
  • Not required, but highly desirable: experience with Ansible, Scala, Hadoop, HBase, Hive
