Job Description
Position Summary:
As a Senior UX Engineer you will use JavaScript and Java to interact and present data from various Data Stores or Web Services to enable valuable insights and analytics for Staples. You will be part of Staples’ Enterprise Data Solutions (EDS) team, which informs Staples’ business decision makers, powers our Data Science systems, and informs our marketing and sales systems. Our mission is to, “Empower decisive informed decisions through timely and accurate data and leading-edge technology.” You will collaborate with data analysts, EDS teammates and other Staples colleagues to help Staples optimize the business and drive personalization for customers. Java experience is preferred, but Staples is willing to train an otherwise very highly qualified candidate who does not have Java development experience.

Primary Responsibilities:
• Develop applications in JavaScript and Java that interact with Database platforms
• Follow standard development processes and best practice
• Participate in design and code reviews
• Perform issue resolution, bug fixes and enhancements
• Provide support for applications once in production
• Become export in and advise customers on data presentation and visualization technologies



Qualifications
Qualifications:
Basic:
• Bachelor’s Degree in Computer Science. a related field, or equivalent work experience
• At least 3 years of experience working with JavaScript and CSS
• Experience with common JS/CSS frameworks (JQuery, Bootstrap, AngularJS)
• Know how to access data and manipulate data from a variety of data sources (Oracle, Couchbase, HIVE)
• Working knowledge of common development tools (Git, Eclipse/IntelliJ, Maven/Gradle, etc.)
• Work experience in an Agile environment
• Ability to learn quickly as you go

Preferred:
• Demonstrated ability creating functional and pleasing User Experience
• Familiarity with Java and Spring framework
• Experience with JavaScript charting frameworks (D3, ChartJS, n3-charts)
• Ability to write scripts in common scripting languages (Bash, Python, Perl)
• Excellent communication skills, verbal and written
• Exude a positive, can-do attitude
