As a Staff Software Engineer for Optimization Technology, you will develop software that enhances GE’s optimization applications and tools in the power generation industry. You will operate in an Agile software development environment as part of a development team consisting of software architects, data scientists, UX designers, product managers, and other developers.
The areas of focus will include: integration of existing .Net-based optimization software with GE’s Predix platform, tools for data-intensive modeling and configuration, and supervisory control application development. As the Staff Software Engineer, you will:
  • Gain and then exploit an in-depth understanding of a large codebase of existing optimization tools, to leverage/evolve/integrate that functionality with GE’s Predix platform
  • Apply principles of SDLC and methodologies like Lean/Agile/XP, CI, Software and Product Security, Scalability, Documentation Practices, refactoring and Testing Techniques
  • Be fluent in core data structures as well as algorithms, and implement them using language of choice when necessary
  • Write code that meets standards and delivers desired functionality using the technology selected for the project
  • Work with Product Managers to understand product requirements & vision
  • Consult with Optimization Application Engineers and other Subject Matter Experts on matters requiring domain (power industry) expertise
