          Description
         This position is primarily responsible for data engineering using T-SQL, SQL Server Integration Services and .NET programming of class libraries and Windows Services in C#. This person will also work closely with engineering management, product managers, and other data engineers to develop new products and improve existing systems for Deluxe Financial Services.
         Accountabilities:
         - Design, document, code and deploy new data products. This includes, but is not limited to the creation of SQL Server Integration Services packages (SSIS), T-SQL stored procedures, functions and views. Development work also includes .NET applications using C# for SQL Server CLR assemblies, class libraries and Windows Services.
         - Participates in both detailed and high level estimates for proposed projects, business value estimations for future features, and time estimates of efforts needed for remediation of software defects, development of new features and increased automation of existing processes.
         - Communicate effectively with members of the Engineering, Quality Assurance, Data Science and Product management teams.
         - Work to ensure a high level of data integrity, performance and cost of ownership standards for all Deluxe software and database products.
         - Develops new knowledge and skills related to technologies within the Hadoop ecosystem. Participates in the design, development and management of processes within this ecosystem.
          Required:
         - Bachelors (or equivalent experience) - Software Engineering.
         - 5 years of experience in full life cycle design and development of data solutions.
         - Contribute thoughtfully on issues surrounding the overall application framework and architecture.
         - Proficiency with SQL Server and subcomponents thereof.
         - Understanding of data flows, data architecture, ETL and processing of structured and unstructured data.
