        Typical Tasks
        As aTechnology Servicesintern, your work could include:
        * Interviewing Town staff to understandtechnology requirements
        * Examining raw data sources to determine what, if any cleanup needs to occur. Work with User community and IT personnelto cleanse data.

        * Making certain the network hardware and software are functional and operating reliably using various network monitoring tools
        * Assisting Service help desk with the following: call center and ticket assignment, backup and off site tape management, software inventory, key codes, and media
        * Engaging in all aspects ofdesktop deployment
        * Troubleshooting network problems and checking equipment logs of network, performing cable testing
        * Developing visualizations of data to aid decision makers.
