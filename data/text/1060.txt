             Data Engineer &#8211; Advanced Analytics Team
             Direct hire no C2C
             No relo
             **This is one of two remaining Data Engineering positions to be filled on a brand new Advanced Predictive Analytics team.**
             About Our Client:
            Our client is the leading provider of web-based monitoring and field automation services to the oil and gas industry.
            In this fast-paced work environment, candidates must be passionate about helping customers, have strong trouble-shooting skills, excellent verbal and written communication skills, be organized, work well in a team as well as independently, demonstrate the ability to learn quickly and be able to manage, prioritize and delegate multiple priorities.
            The oil &amp; gas industry and its move to turn data into meaningful business decisions, is evolving into the Cloud. Building industry leading Cloud technical solutions requires experimenting with new technologies and taking ownership of developing enterprise solutions. The Data/ETL Engineer will be part of a Predictive Analytics team that will answer the most critical oil &amp; gas questions the industry faces today.
             Reports To:
            Director of the Predictive Analytics Team
             Responsibilities:
            Develop Data Warehouse and ETL procedures using SSIS, Azure Data Factory, Azure Data Warehouse
              Create data integration and migration technical solutions for SQL Azure databases and Azure Data Warehouse;
              Performs technical walk-through as needed to communicate design/coded solution and to seek input from team lead and members;
              Develops ETL processes using various tools to perform control tables, error logging, auditing and data quality; and Implements data cleanup procedures, transformations, scripts, stored procedures, and execution of test plans for loading the data successfully into the targets.
            Data Modeling and Requirements Analysis
              Prepare DB logical and physical models based on requirements;
              Review and obtain sign off on data models from architecture and other supporting organization Database Design Hierarchy;
              Work closely with the DBA team and communicate any hardware/technical requirements;
              Work closely with Data Science team to provide datasets necessary for experiments and creating models;
              Understand, articulate and present business requirements into technical solutions;
              Analyze and profile current data to ensure no requirements are missed; and
              Work with quality assurance team to build test plans to validate that new systems meet the business requirements.
             Desired Skills and Experience
             Requirements for the Position
              SQL Server 2012 and above, SQL Azure, SQL Server Integration Services, Data Warehouse;
              Sr. level understanding of warehousing architectures including ETL design, staging, transformations, delta/change data capture, star-schemas, cubes, and history loading;
              Experience leveraging ETL frameworks to develop scalable solutions using SSIS. Design and develop SSIS packages, configuration files, stored procedures, tables, views and functions;
              Understand database platform layouts and configurations;
              Understand reusability, parameterization and workflow design;
              Experience working in an agile team environment. Understanding of code promotion, code turnover to operations, version control, and QA procedures. Conduct code walkthroughs, peer reviews, and produce technical documentation; and
              Strong Data Modeling skills to include data quality, source systems analysis, business rules validation, source target, mapping design, performance tuning and high volume data loads.
             Desired additional experience/skills
              Excellent communication skills
              SQL Server Reporting Services, SQL Server Analysis Services, SQL Server Data Quality Services;
              Experience in using Cloud based managed services for data warehousing/analytics in Microsoft Azure (Azure Data Lake, Azure Data Lake Analytics, Azure Data factory, USQL, Stream Analytics, HDInsight, etc.) is highly desired.
              Experience in a professional services environment with exposure to senior functional leaders
              Experience in Oil &amp; Gas or other industrial-related data analytics field preferred, and highly desired
