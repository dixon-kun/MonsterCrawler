POSITION SUMMARY:

ADP's Innovation Labs is searching for a successful Sr. UI Developer to join our high-performance team. The successful candidate will have a strong command of software programming, latest web technologies, and front-end frameworks. This is a senior position where the candidate's primary responsibility will be developing user-focused, innovative, scalable and highly interactive web applications.

Responsibilities

* Design, implement and own responsive, scalable, and robust cross-platform and cross-browser-compatible web applications

* Develop user and user experience-focused interactions

* Support, test, and debug production software and operations

* Report progress and status of assigned tasks to project teams and managers

* Work directly with our designers and back-end developers to continuously improve our web applications

* Iterate quickly based on feedback

Qualifications

* Expert in JavaScript, HTML5 and CSS3 development (Required)

* In-depth experience with React or a similar front-end framework (Required)

* Experience consuming RESTful web services via AJAX

* Experience with JSON (Required)

* Passion for great user experience (Required)

* In-depth experience with Chrome Developer Tools (or similar) for profiling and debugging front-end web applications (Highly desired)

* In-depth experience with modern front-end build tools and package management, like webpack, Bower, browserify, Grunt or Gulp (Highly desired)

* Experience with open source software (Required)

* Strong understanding of object-oriented design and MVC patterns (Highly desired)

* Excellent understanding of web security practices (Highly desired)

* Skills and experience with other languages and platforms, i.e. NodeJS, Java, PHP, or Python (Nice to have)

* Experience with both SQL and NoSQL databases, preferably MongoDB (Nice to have)

* Experience developing and deploying across multiple platforms - Windows, Linux and UNIX (Nice to have)

* Strong understanding of agile development and continuous integration (Highly desired)

* Strong written and verbal communication skills (Required)

* Bachelor's degree in Computer Science, Engineering or related discipline (Highly desirable)

* 5+ years of hands-on experience developing Web applications (Required)
