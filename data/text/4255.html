Neuroscience Data Scientist – Boston, MA

Join the Nielsen Consumer Neuroscience Team as a Neuroscience Data Scientist to apply your analytical and programming skills in the exciting field of consumer neuroscience.

Primary responsibilities:

-      Process and analyze data from multiple signals, including eye tracking, biometrics, self-report, facial coding and behavioral coding

-      Run quality control checks and data mining on biometric data streams

-      Provide support for streamlining and operationalizing data workflow

-      Run statistics such as, but not limited to, linear regressions, correlations, ANOVAs, t-tests.

-      Flag and relay messages to avert potential analytical issues in a timely manner

-      Assist/support data collection (as needed)

Qualifications

Experience/ Qualifications:

-      Bachelor’s degree required; Graduate degree a plus

-      Minimum 3-4 years’ experience with data analysis in a business setting

-      Experience with signal processing of psychological and/or physiological measures, ideally has lab experience

-      Proficiency with any of the following programming languages: MatLab, Python, R, C (other programming knowledge/experience a plus)

-      Background in basic statistics; SPSS, SAS experience a plus

-      Advanced user of Excel/PowerPoint; VBA proficient

-      Familiarity with basic video editing software, Photoshop, html, website creation

Skills:

-      Possesses strong analytical skills

-      Able to work as part of a team & support other members, as well as work autonomously

-      Is quick learner, able to adapt to using new software on the fly

-      Displays maturity and professionalism

-      Able to adapt easily to changing conditions/schedules and client needs

-      Has great attention to detail, takes pride in quality of their work

-      Willing to travel occasionally for client work if needed
