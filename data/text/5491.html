Scope of This Position:

* Build machine learning models aimed towards unlocking value from data sources

* Work with divisional stakeholders to validate machine learning models

* Build a workflow for on-going evaluation and refinement of machine learning models deployed in digital products

* Develop KPIs for tracking, baselining and monetisation of machine learning models developed

Job Description

Responsibilities:

* Build machine learning models using supervised and unsupervised modeling techniques

* Working under the guidance of project leads, help manage team projects via task scheduling and tracking, issue management, status reporting, budget management, and stakeholder communication

* Develop statistical, mathematical and predictive models using strategic business understanding to build the algorithms necessary to ask the right questions and find the right answers

* Explore and examine data from multiple disparate sources in order to identify, provide reporting on, and analyze trends in the KPIs as well as regular reporting for assigned departments

* Provide technical analysis to prepare specifications for visual analytics and reporting needs – assist with QA and testing of new dash boarding/widgets

* Explore and examine innovative ways to optimize the 'cost-to-analyze' though an intelligent application of machine learning toolsets

Qualifications

Required:

* Master's Degree Required

* 5-8 years of related experience

* Data science architecture and role of machine learning

* Machine learning

* Supervised and unsupervised machine learning model trainings
