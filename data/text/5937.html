SUMMARY

The Release Manager provides valuable insights, management, and ensures that all products are released to production without adverse effect to clients while maintaining the technical integrity of applications. This individual ensures release implementations are approved and that only correct, authorized, and tested code versions are installed in production environments.

The Release Manager will have the ability to troubleshoot and resolve failures to production code promotions, will support continuous improvement of the release management process to meet business needs, and will maintain current release management tools, documents, and reports in an agile scrum development environment.

This individual will be committed to developing innovative enterprise release management solutions and releasing it quickly, efficiently and with increasing quality to knock-the-socks-off of our client user experience. The individual will play a critical role in securing ADP's success by ensuring development operations are fully optimized to meet ADP and client needs.

The ideal candidate would have experience spanning multiple disciplines, including application development, infrastructure engineering and operations monitoring, configuration management, and release management.

WHAT WILL YOU DO?

* Guide initiatives to align with the overall application portfolio and core platform blueprint and manages the evolution of the application portfolio from a release management vantage point.

* Manages Release Management function in the successful implementation of software code for all applications

* Manages the development of procedures and policies related to software release of applications during the development and maintenance product life cycle

* Interfaces with key software providers to establish release management guidelines

* Ensures that all software releases meet time constraints while ensuring that specifications are met and delivered.

* Understands and manages large scale application deployments

* Develops build and release engineering plans

* Shows proven ability to develop automated continuous delivery and deployment solutions

* Works with development teams and others to create infrastructure as code using Docker, Puppet, Chef, or similar

* Establish enterprise-wide continuous integration and continuous delivery capability using GitHub, Bitbucket, Subversion, SonaType Nexus, and Jenkins

* Support traditional release management concepts while adopting modern DevOps release principles and best practices

ABOUT YOU

* 7+ years of experience working in Release Management

* 5+ years of experience with large commercial application delivery

* 5+ years leading people and managing team workloads

* 5+ years of experience managing projects

* 3+ years of experience working in Agile Scrum software development environment

* 3+ years of continuous integration/continuous delivery experience in agile scrum environment

* 2+ years of experience with source code management system configuration and administration using Git, GitHub, Bitbucket, or similar

* Proficient with Windows, Unix, and Linux operating systems

Preferred Skills:

* Experience with Docker and any scripting language is a big plus

* Excellent verbal & written communication skills and be able to communicate and adapt effectively to different audiences

* Experience in office and collaboration tools (SharePoint, Rally, Confluence, Trello) is a plus

* Software programming experience in Java/JavaScript/C#/HTML5/CSS3 is a plus

* Process automation experience in continuous integration and continuous delivery using Git, Bitbucket, Docker, Jenkins is a big plus

* Experience with build tools Ant and Maven is a plus
