        Description:
        CACI seeks experienced Data Architect with a background in biometrics, cloud architect, architecting new application solutions, data migrations, information technology and database management that possess an active DoD Secret clearance with experience working in a Department of Defense environment. This position is a senior level highly technical position with responsibility to lead the creation and use of data deliverables in support of Automated Biometrics Information System (ABIS). Data deliverables will include requirements, data analysis including profiling, modeling, data flow diagrams, data sourcing plans, data management, database architecture and data transformation strategies. is a demanding role that requires deep technical knowledge and adaptability with hands on execution. You must have excellent communication skills, both verbal and written, and be a strategic thinker.
        Duties and Responsibilities:
        The Data Architect is responsible for designing, developing, and driving enterprise data integration and migration strategy, methodology, and standards that shape the client's modernization programs and initiatives. Design a robust technical solution to migrate and integrate data from multiple applications, so that ongoing data migrations and integrations can be re-factored with minimal rework. The position is responsible for data sharing interoperability support and portal application administration and maintenance. Provide technical, strategic guidance on the development of metadata tags, Document Type Definitions (DTD), and schemas using appropriate technologies for representation such as HTML and XML. Ensure that metadata and data standards and definitions will support DoD environment and system implementations requirements for sharing data. The position interfaces with clients to comprehend business needs, map data requirements, unify vocabularies, ontologies, and taxonomies, and develop data analytics architecture to drive a future state data platform, applying innovative data architecture principles and techniques. Develop required technical documentation, including a data migration plan, data mapping specifications, data extraction file layout and specifications, data migration, and data verification protocols. Manage and coordinate data integration activities with the delivery team, including tasks and timelines and the execution of data cleansing and standardization activities to be implemented. Develops project standards and processes and uses automated development tools; Applies reverse engineering and re-engineering disciplines to develop migration strategic and planning documents. Support the project teams during the user acceptance testing activities and assist with the development of a post-deployment data maintenance plan.
