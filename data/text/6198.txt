Job Description
As a Data Scientist with 1904labs, you will be responsible for working with various environment tools and data like spark to create and test machine learning algorithms. Once tested and verified this person will work with development teams to implement the algorithm within an orchestrated set of microservices.
Requirements:
Computer Science: Background in computer science and Math with experience understanding statistical models
OO Programming: Experience working with a language suited for machine learning systems such as Python, R, Java or Scala REST Services Development
Machine Learning Analysis: Experience evaluating and applying various models such as decision tree, nearest neighbor, neural net, support vector machine, an ensemble of multiple models, etc...
 Analytics: [1 of] Spark, Spark Streaming, Flink, DataBricks, Samza, MapReduce
 Tools: Experience using tools like &#8220;Jupyter Notebook&#8221; to perform Analysis activities and test / evaluate data models and algorithms.
 Attitude / Aptitude: Candidates must be adaptable and have the drive to learn new technologies and frameworks to support R&amp;D interests
 Agile: Experience working in an Agile team oriented environment.
