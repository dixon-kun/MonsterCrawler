Technical Writer - Big Data at eXcell

Cambridge 02142

Report this job

About the Job

Did you know that our employee benefits start on the first day of employment? Join e X cell and enjoy our generous, DayOne Benefits(SM)!

We are looking for a Technical Writer who can work with the data science team in Cambridge, MA to create well-crafted Machine Learning blog posts, Conference Papers, Solution Documentation, etc. We need a candidate with awareness of big data, cloud technologies, and publishing documentation in that space. Data Science & machine learning experience would be an added advantage.

Overview:
  • Research, design, develop, and write technical information online or in print for a variety of audiences and purposes.
  • Produce documentation that complies with documentation plan requirements.
  • Develop and test user scenarios that show how to use the product being documented.
  • Obtain technical information from variety of project owners; convert specific feedback into documentation.
  • Respond to editorial and reviewer feedback.
Qualifications:
  • 4-5 years of experience
  • Technical documentation work in a tech company
  • Should be a self-starter, manage their own work, prioritizing, updating status, reporting, creating visibility.
  • Produced customer / public documentation
  • Exposure to big data and cloud technologies, data science, machine learning
  • Portfolio preferred
W2 only, no Corp to Corp.
