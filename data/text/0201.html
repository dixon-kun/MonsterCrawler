**Python DevOps Developer ** The Python Devops Developer will be a member of the Analytics and Innovation team. We productize big data and data science reports and process the raw data for our public relations and advertising customers. We’re looking for a developer to focus on DevOps, but with a willingness to branch out and do some ad-hoc data science stuff. **What you’ll be doing:** **Core- Python web app development.**Maintaining one or more large scale complex data-centric Django applications, keeping up with 3rd party API changes, AWS changes, bug fixing and feature requests. * Writing automated tests to demonstrate provable, on-going, correctness of code **Core- Database Development.** Writing ad-hoc SQL queries, Django-ORM queries * Work with a Redshift database cluster (similar to Postgres) **Core- Data Science Development.** New data science reports involving automating basic long running tasks that use scikit-learn, NLTK. **Core- Cloud Development or “DevOps”.**Scripting “devops” tasks. Because application has significant cloud could components, you’ll be expected to skill-up on a variety of server products enough to install, troubleshoot on your workstation and the development servers and then stand up test cloud servers. **Occasional and Incidental** * A few time a year using tools and python to perform ETL tasks (extracting, transforming and loading large quantities of data) * May be involved with incidental support desk work, demoing, etc. * Some Postgres specific administrative tasks, such as performance tuning, backups and restores **_Team Environment and Perks_** Team is doing sincere Scrum, with daily scrum, Jira, frequent small releases and defined roles. As a developer, you’ll have significant autonomy with respect to how get your job done. We use a variety of techniques to ensure quality, including automated unit testing, integration testing, linting and automated builds. Mac Pro workstation, video training, annual training budget, top of the line tooling, such as PyCharm, GitHub, QuantifiedCode and others, quiet work environment, metro accessible, support for occasional work-from-home (up to 20%), flexible workhours **Key Skills and Technology Inventory** ***Must Have***: * Bachelors degree or equivalent experience and portfolio * 3+ years database development, using SQL (any dialect) * 3+ years python development (or 5 years experience in Ruby, JavaScript, Node or other dynamic programming language) * Full stack experience supporting an application with client side, server side and database components, with any technology stack * Demonstrated ability to write batch, powershell or bash scripts. ***Nice to Have:*** * Full stack experience supporting an application with client side, server side and database components, especially Angular, CSS, Django, Celery and Postgres * AWS experience * Experience with unit testing * Experience with Mac, Linux ***About Burson-Marsteller*** Burson-Marsteller, established in 1953, is a leading global public relations and communications firm. It provides clients with strategic thinking and program execution across a full range of public relations, public affairs, reputation and crisis management, advertising and digital strategies. The firm’s seamless worldwide network consists of 73 offices and 85 affiliate offices, together operating in 110 countries across six continents. Burson-Marsteller is a unit of WPP, the world’s leading communications services network. Burson-Marsteller is an Equal Opportunity Employer, EEO, AAE, MF/Disability/Vet, and we are proud of our diverse workforce. *Location:* US - Washington, DC
