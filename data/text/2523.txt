         Description:
        Leidos is looking for a Software Developer for a position in Chantilly, VA. This position requires an active TS/SCI with Polygraph in order to be considered.
         The division provides support in the areas of data extraction, transformation and load (ETL), data mapping, data extraction, analytical support, operational support, database support, application development and associated maintenance of production systems. Team members work in a multitasking, quick-paced, dynamic, process-improvement environment that requires experience with the principles of biometric technology, large-scale (Terabytes) database and application development, large-scale file manipulation, data modeling, data mapping, data testing, data quality, and documentation preparation. The division directly supports data scientists and analysts by designing and engineering systems to meet exponential increases in digital data.  This position is for a specialist in data Extract, Transform, Load (ETL) processing and related software development, data modeling, testing, and technical documentation. The ETL data developer will work within a team of ETLers to process datasets from a wide variety of sources into a large-scale information system. The developer will work with priorities established by a team lead and report on progress toward delivery of datasets through an established ETL workflow.
